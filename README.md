## Key Features

* User can create/delete/update to-do
* User can set priority for to-do
* User can set to-do completed
* User can group to-do for one named project

## How To Use

```bash
# Clone this repository
$ git clone https://github.com/TashfeenRao/todo-js.git

# Go into the repository
$ cd todo-js

# Install dependencies
$ npm install

# Run the app
$ npm start
```
